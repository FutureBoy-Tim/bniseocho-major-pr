# 검색 노출 + 확산 가이드

> 이 문서는 비기술자가 따라 할 수 있도록 작성됨. 클로드가 기술 작업은 끝냈고, 여기 적힌 단계는 **사람이 직접 외부 사이트에서 해야 하는 액션**들.

---

## 📍 현재 상태 (클로드 완료)

- ✅ `sitemap.xml` (39명 멤버 페이지 모두 포함)
- ✅ `robots.txt` (네이버·구글·다음 크롤러 허용)
- ✅ JSON-LD 구조화 데이터 (Organization · WebSite · Person)
- ✅ `canonical` URL 설정
- ✅ OG 메타 태그 (카톡 라지 카드)
- ⏳ 검색엔진 사이트 소유 확인 메타 (자리만 만들어둠 — 사용자 인증 코드 받은 후 교체 필요)

## 1단계 · 네이버 사이트 등록 (한국 검색 60% 점유 — 가장 중요)

**소요 시간**: 15분 / **빈도**: 1회

### 1.1 네이버 서치어드바이저 가입

1. [https://searchadvisor.naver.com/](https://searchadvisor.naver.com/) 접속
2. 우측 상단 **로그인** → 본인 네이버 계정으로 로그인
3. 첫 진입 시 약관 동의

### 1.2 사이트 등록

1. 상단 메뉴 **"웹마스터 도구"** 클릭
2. 사이트 추가 입력란에 다음 입력:
   ```
   https://futureboy-tim.github.io/bniseocho-major-pr/
   ```
3. **추가** 버튼 클릭

### 1.3 사이트 소유 확인 (메타 태그 방식 추천)

1. **HTML 태그** 탭 선택
2. 화면에 표시되는 메타 태그가 다음과 비슷한 형태:
   ```html
   <meta name="naver-site-verification" content="abc123def456..." />
   ```
3. **`content="..."` 안의 값(`abc123def456...` 부분)을 복사**
4. 클로드에게 한 줄로 부탁:
   > "네이버 인증 코드는 `abc123def456...` 야. 박아줘"
5. 클로드가 [index.html](index.html)·[member.html](member.html)의 `REPLACE_WITH_NAVER_CODE` 부분을 자동 교체 + push
6. 1~2분 후 네이버에서 **확인** 버튼 클릭

### 1.4 사이트맵 제출

1. 좌측 메뉴 **요청 → 사이트맵 제출**
2. 입력란에:
   ```
   sitemap.xml
   ```
3. **확인** 클릭
4. 며칠 안에 39명 멤버 페이지가 네이버 검색에 인덱싱됨

## 2단계 · 구글 서치 콘솔 등록 (글로벌 + 한국 30%)

**소요 시간**: 10분 / **빈도**: 1회

### 2.1 등록

1. [https://search.google.com/search-console](https://search.google.com/search-console) 접속
2. 본인 구글 계정으로 로그인
3. **속성 추가** → **URL 접두어** 선택
4. 다음 URL 입력:
   ```
   https://futureboy-tim.github.io/bniseocho-major-pr/
   ```
5. **계속** 클릭

### 2.2 소유권 확인

1. **기타 확인 방법** 펼치기 → **HTML 태그**
2. 표시되는 메타 태그:
   ```html
   <meta name="google-site-verification" content="xyz789abc..." />
   ```
3. `content="..."` 안의 값 복사
4. 클로드에게:
   > "구글 인증 코드는 `xyz789abc...` 야. 박아줘"
5. 자동 교체 + push 후 **확인** 클릭

### 2.3 사이트맵 제출

1. 좌측 메뉴 **Sitemaps**
2. 입력란에:
   ```
   sitemap.xml
   ```
3. **제출** 클릭

## 4단계 · (선택) 자체 도메인 연결

GitHub 도메인(`futureboy-tim.github.io/bniseocho-major-pr/`)은 길고 기억하기 어려움. **자체 도메인**으로 바꾸면 신뢰도·확산력·전문성이 올라감.

### 4.1 도메인 구입 (가비아 추천)

1. [https://www.gabia.com/](https://www.gabia.com/) 접속
2. 검색창에 원하는 도메인 입력 (예: `bniseocho`, `bnimajor`)
3. 추천 도메인 (예시):
   - `bniseocho.kr` — 약 2.2만원/년
   - `bniseocho.com` — 약 1.4만원/년
   - `bnimajor.kr` — 약 2.2만원/년
4. 결제 완료

### 4.2 도메인 → GitHub Pages 연결

도메인 구입 후 클로드에게:
> "도메인 `bniseocho.kr` 샀어. GitHub Pages에 연결해줘"

클로드가 자동 처리:
- `CNAME` 파일 생성 (도메인 명시)
- GitHub Pages 설정 변경
- 모든 OG·canonical·sitemap URL 새 도메인으로 일괄 교체
- 가비아 DNS 설정 안내 (CNAME 또는 A 레코드)

사용자가 가비아에서 해야 할 일 (안내받음):
1. 가비아 → 도메인 관리 → DNS 설정
2. 다음 4개 A 레코드 추가:
   ```
   185.199.108.153
   185.199.109.153
   185.199.110.153
   185.199.111.153
   ```
3. 또는 CNAME: `futureboy-tim.github.io` 1개 (서브도메인 사용 시)
4. 저장 → 30분~24시간 내 DNS 반영

### 4.3 결과

- `https://bniseocho.kr` 같은 깔끔한 URL
- 무료 SSL (HTTPS) 자동 적용
- 이전 GitHub URL도 자동 리다이렉트

## 5단계 · 확산 운영 룰 (39명 멤버 활용 — 가장 강력한 엔진)

> SEO보다 빠르고 강력. 39명 = 확산 채널 39개.

### 5.1 매주 피처드 멤버 본인 SNS 공유 (의무화 권장)

**룰**:
- 본인이 피처드되는 주에는 다음 채널 중 **3개 이상**에 게시:
  - 카카오톡 프로필 메시지
  - 페이스북 개인 페이지
  - 인스타그램 스토리 또는 피드
  - 링크드인
  - 본인 회사 블로그 / 홈페이지
- 게시 카피는 **이미 사이트가 자동 생성**:
  - 사이트 → 우측 상단 "📢 외부에 공유" 버튼
  - 4가지 톤(친근/전문/공식/짧은) 중 선택해서 복사
- 게시 후 챕터 단톡방에 인증샷 1장

**효과**: 피처드 멤버의 인적 네트워크에 직접 노출 → 평균 멤버 1명당 SNS 친구·팔로워 500명 = **39명 × 500 = 약 19,500명 잠재 노출**

### 5.2 명함 QR 코드 추가 (오프라인 확산)

**룰**:
- 다음 명함 인쇄 시 QR 코드 추가 (사이트 URL 연결)
- BNI 멤버는 매주 신규 명함 교환 다수 발생 → 받은 사람이 즉시 사이트 방문
- QR 위치: 명함 뒷면 또는 모서리

**QR 코드 만들기 (무료)**:
1. [https://qr.naver.com/](https://qr.naver.com/) 또는 [https://www.qr-code-generator.com/](https://www.qr-code-generator.com/)
2. URL 입력:
   ```
   https://futureboy-tim.github.io/bniseocho-major-pr/
   ```
   (자체 도메인 연결 후에는 새 도메인 사용)
3. 다운로드 → 명함 디자이너에게 전달

**효과**: BNI 멤버 39명 × 매주 평균 20명 명함 교환 = **주당 780회, 연간 4만 회 노출**

### 5.3 챕터 공식 카카오톡 채널 (선택)

**개설**:
1. [카카오톡 채널 관리자센터](https://center-pf.kakao.com/login) 가입
2. 채널 개설 → "BNI 서초 메이저"
3. 프로필 이미지: 챕터 로고 또는 이번 주 피처드 인포그래픽
4. 사이트 URL 등록

**운영**:
- 친구 추가한 외부 사람에게 매주 피처드 자동 발송
- BNI 멤버 39명 모두 친구 추가 (기본 39명 채널 도달)
- 멤버들이 자기 네트워크에 채널 공유

### 5.4 인스타그램 활용 (1:1 인포그래픽 그대로 사용 가능)

**계정 개설**:
- 핸들: `@bniseocho_major` 또는 `@bni.seocho`
- 프로필: BNI 로고, 사이트 URL 링크 (Linktree 또는 직접)

**콘텐츠 룰**:
- 매주 월요일: 피처드 멤버 인포그래픽 (1:1 정사각이라 그대로 게시)
- 캡션: 사이트의 자동 생성 카피 사용
- 해시태그: `#BNI #BNI서초 #서초메이저 #비즈니스리퍼럴 #서초사장 #서초CEO`
- 스토리에 사이트 링크 첨부 (계정 1만 팔로워 이상 시 스와이프업, 그 미만은 링크 스티커)

### 5.5 페이스북·인스타 광고 (예산 들어가는 옵션)

**시점**: SEO + 자체 도메인 안정화 후

**예산 가이드**:
- 시범 운영: 월 30만원 (일 1만원)
- 안정 운영: 월 50~100만원

**타겟 세팅**:
- 위치: 서초구·강남구·반포·송파 반경 5km
- 연령: 35~55세
- 관심사: 비즈니스, 사업가, 컨설팅, 세무, 법률, 의료
- 행동: "사업체 소유자" 또는 "고소득"

**소재**:
- 매주 피처드 멤버 인포그래픽 + 짧은 카피 + 사이트 링크
- 카피 예시: "병원 원장님이 만나야 할 검증된 절세 전문가, 매주 한 분씩 만나보세요"

## 6단계 · 효과 측정

### 6.1 구글 애널리틱스 (선택)

방문자 수·체류 시간·인기 멤버 페이지 측정.

1. [https://analytics.google.com/](https://analytics.google.com/) 가입
2. **속성 만들기** → 사이트 정보 입력
3. **측정 ID** (`G-XXXXXXX`) 받기
4. 클로드에게:
   > "GA 측정 ID는 `G-XXXXXXX` 야. 박아줘"
5. 자동 박힘 → 다음날부터 데이터 수집

### 6.2 네이버 검색 노출

- 네이버 서치어드바이저 → **리포트 → 검색 노출**
- 어떤 키워드로 검색됐는지, 클릭 수, 노출 수 확인

### 6.3 카카오톡 공유 추적 (한계 있음)

- 카카오는 외부 사이트 방문 추적이 제한적
- GA의 referrer 확인으로 카톡 유입 일부 파악

## 💡 운영 팁

- **검색 노출은 시간이 걸림**: 네이버는 보통 2~4주, 구글은 1~2주 후부터 본격 인덱싱.
- **확산이 SEO보다 빠름**: 첫 한 달은 39명이 자기 네트워크에 공유하는 것에 집중.
- **매주 갱신 자체가 SEO**: 검색엔진은 "자주 업데이트되는 사이트"를 좋아함. 매주 피처드 변경 = 자연스러운 SEO 신호.
- **카톡 라지 카드는 캐시 필수**: 새 멤버 공개 후 [카카오 캐시 초기화](https://developers.kakao.com/tool/clear/og)에서 URL 캐시 삭제 1회.
